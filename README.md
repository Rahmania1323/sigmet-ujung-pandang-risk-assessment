# SIGMET Risk Assessment in Ujung Pandang FIR Airspace

This repository contains Python-based analysis code, figures, and supporting materials for the study:

**“A Multi-Year Risk Assessment of SIGMET Evolution in Ujung Pandang FIR Airspace: Reconstructing Fragmented Spatiotemporal Hazards for Aviation Decision Support (2019–2025)”**

---

## 📄 Description

This repository contains Python-based analysis for reconstructing and evaluating SIGMET-related aviation weather hazards in the Ujung Pandang FIR during 2019–2025. The study focuses on temporal trends, spatial distribution, and risk characterization of SIGMET events to support aviation decision-making.
